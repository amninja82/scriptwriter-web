## 📱 使用教程

### 第一次使用

1. **启动程序**
   ```bash
   python scriptwriter_gui.py
   ```

2. **设置 API Token**
   - 点击左侧 "⚙️ 设置 Token" 按钮
   - 输入您的 Coze API Token
   - Token 获取地址：https://coze.com/user/pat
   - Token 格式：`pat_xxxxxxxxxxxxx`

3. **等待连接**
   - 程序会自动连接到智能体
   - 连接成功后状态栏显示 "🟢 已连接"

### 创建项目

1. 点击左侧 "➕ 新建项目" 按钮
2. 输入项目名称
3. 选择项目类型
4. 输入项目描述
5. 点击 "创建"

### 与智能体对话

1. 在右下角输入框输入消息
2. 按 `Ctrl+Enter` 发送
3. 等待智能体回复
4. 对话自动保存

### 联网搜索

1. 点击左侧 "🔍 联网搜索" 按钮
2. 输入搜索内容（如"悬疑剧开篇技巧"）
3. 点击 "🔍 开始搜索"
4. 智能体会自动：
   - 联网搜索相关内容
   - AI 分析内容类型
   - 自动分类保存到知识库

### 导出对话

1. 点击左侧 "💾 导出对话" 按钮
2. 选择保存位置
3. 对话会保存为文本文件

## ⚠️ 注意事项

### 1. API Token 安全

- Token 保存在本地 JSON 文件中
- 请妥善保管，不要泄露给他人
- 定期更换 Token

### 2. 网络连接

- 需要网络连接才能与智能体对话
- 联网搜索需要稳定的网络
- 建议使用稳定的网络环境

### 3. 对话存储

- 对话历史保存在本地
- 删除程序不会丢失对话（数据文件独立）
- 建议定期备份 `scriptwriter_gui_data.json`

### 4. 跨平台

- GUI 版本支持 Windows、macOS、Linux
- 不同平台可能需要不同的打包方式

## 🆘 常见问题

### Q1: 程序启动后无法连接？

**A**：
1. 检查网络连接
2. 确认 Token 是否正确
3. 检查 Token 是否过期

### Q2: 联网搜索失败？

**A**：
1. 检查网络连接
2. 确认搜索内容是否合适
3. 尝试重新搜索

### Q3: 项目对话丢失？

**A**：
1. 检查 `scriptwriter_gui_data.json` 是否存在
2. 确认文件未被删除或损坏
3. 尝试从备份恢复

### Q4: 如何批量导入对话？

**A**：
目前不支持批量导入，建议：
1. 使用导出功能备份
2. 需要时手动导入到 JSON 文件

### Q5: 可以离线使用吗？

**A**：
- 查看历史对话：可以离线
- 与智能体对话：需要在线
- 联网搜索：需要在线
