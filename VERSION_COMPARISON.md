# 多智能体编剧系统 - 版本对比与选择指南

## 📦 可用版本

### 1. GUI 版本（图形界面）⭐ 推荐

**文件**：`scriptwriter_gui.py`

**特点**：
- ✅ 图形界面，操作直观
- ✅ 项目管理，创建多个项目
- ✅ 对话历史本地存储
- ✅ 联网搜索，自动保存到知识库
- ✅ 支持导出对话记录
- ✅ 可打包为 exe，无需 Python 环境

**适合人群**：
- 喜欢图形界面的用户
- 需要管理多个项目的用户
- 需要本地保存对话历史的用户

**快速开始**：
```bash
# 直接运行
python scriptwriter_gui.py

# 或打包为 exe
build_windows_gui.bat
```

## 🎯 知识库说明

### 知识库来源

所有版本都使用**相同的知识库**：

#### 1. Coze 平台知识库（云端）

**位置**：Coze 平台

**内容**：
- 编剧技巧
- 题材分析
- 故事钩子
- 情绪价值
- 冲突设计
- 影视案例

**访问方式**：智能体自动调用

#### 2. 联网搜索 + 自动保存

**功能**：
- 联网搜索最新内容
- AI 自动分析类型
- 自动分类保存到知识库

**使用方式**：
- GUI 版本：点击 "🔍 联网搜索" 按钮
- CLI 版本：输入 "搜索XX并保存到知识库"
- Coze API 版本：直接对话

**保存位置**：Coze 平台知识库
