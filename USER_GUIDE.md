## 🎬 完整功能说明

### 一、知识库管理

#### 1. 上传本地文件

**功能**: 将本地文件（.txt, .md, .docx等）上传到知识库

**使用方法**:
```
你：上传这个文件 /path/to/file.txt
我：✅ 文件上传成功！文档ID: xxx
```

**支持工具**:
- `upload_text_file_to_knowledge` - 上传单个文本文件
- `batch_upload_files_to_knowledge` - 批量上传多个文件
- `upload_url_to_knowledge` - 上传 URL 内容

**示例**:
```
你：批量上传这些文件
    - /home/user/history.txt
    - /home/user/characters.md
我：📦 批量上传完成
    成功: 2 个文件
    失败: 0 个文件
```

#### 2. 搜索知识库

**功能**: 从知识库中检索相关内容

**使用方法**:
```
你：搜索古代刑法相关知识
我：[返回相关的知识库内容]
```

**示例**:
```
你：我的剧本需要关于三国时期的历史资料
我：[搜索并返回相关内容]
你：把这些资料保存到知识库
我：✅ 已保存到三国资料数据集
```

#### 3. 在线搜索

**功能**: 搜索网络并自动保存到知识库

**使用方法**:
```
你：搜索古代宫廷制度并保存
我：🌐 正在搜索... [返回搜索结果]
    ✅ 已保存到知识库
```

### 三、创意引导

当你的创意比较模糊时，系统会通过对话逐步引导你完善需求。

#### 引导流程

系统会依次询问以下问题：

1. **创意构思** - 你的初步想法
2. **题材类型** - 悬疑/爱情/科幻/历史等
3. **受众群体** - 目标观众年龄段
4. **集数/时长** - 项目规模
5. **风格偏好** - 悬疑紧张/温馨治愈等
6. **对标作品** - 参考作品

#### 使用示例

**场景1：只有初步想法**
```
你：我想写一个关于时空穿越的剧
我：很好！这个剧属于哪种类型？
    - 科幻
    - 爱情
    - 悬疑
    - 其他
你：科幻爱情
我：明白了！那目标观众是谁呢？
    - 18-25岁
    - 25-35岁
    - 35-50岁
    - 全年龄层
你：25-35岁的职场女性
我：收到！这个剧本的规模是？
    ...
```

**场景2：跳过某些问题**
```
你：我不想设置对标作品
我：好的，跳过对标作品。
    ✅ 信息收集完成！正在生成需求报告...
```

#### 查看已收集信息

```
你：看看我刚才填了什么
我：📋 需求采集摘要
    **创意构思**: 时空穿越的故事
    **题材类型**: 科幻爱情
    **受众群体**: 25-35岁职场女性
    **集数/时长**: 20集
    **风格偏好**: 浪漫唯美
    **对标作品**: 无
```

## 📝 使用技巧

### 技巧1：知识库先行

在创作前，先建立知识库：
```
你：先上传一些历史资料
    - 三国历史.txt
    - 古代官职制度.md
    - 宫廷礼仪手册.docx
```

### 技巧2：模糊创意使用引导

如果只有模糊想法，直接使用创意引导：
```
你：我想写个剧
我：🎬 创意引导已启动
    请告诉我你的创作创意，可以是一句话、一个想法...
```

### 技巧3：随时查看进度

创作过程中随时检查进度：
```
你：现在到哪一步了
我：[显示当前进度]
```

### 技巧4：分批完善需求

如果某些细节不确定，可以先跳过：
```
你：对标作品先不填
我：好的，跳过对标作品。
```

## ❓ 常见问题

### Q1: 如何上传 Word 文档？

A: 先转换为 .txt 或 .md 格式，再上传：
```
你：上传 /path/to/document.txt
```

### Q2: 知识库支持哪些格式？

A: 支持纯文本格式：
- .txt
- .md
- .json
- .csv

### Q3: 创作需要多长时间？

A: 取决于剧本规模：
- 短剧（10集）：约 3-5 分钟
- 中篇（20集）：约 8-12 分钟
- 长篇（40+集）：约 15-20 分钟

### Q4: 如何修改已创作的内容？

A:
```
你：修改剧本，把结局改成开放式
我：🔄 正在修改...
    ✅ 修改完成！v2
```

### Q5: 如何导出剧本？

A: 系统会自动保存剧本，可以通过以下方式获取：
```
你：导出剧本
我：✅ 剧本已保存，可通过项目文件获取
```
