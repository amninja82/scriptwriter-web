# 剧本创作工作流 - 标准流程

## 🎬 完整创作流程（8节点版本）

### 阶段一：前期准备（节点1-3）

#### 📋 节点1：需求解析
**负责 Agent**：策划师 Agent

**工作内容**：
- 解析用户输入的创作需求
- 提取核心要素
- 初步确定创作方向

**输出**：
- 题材类型
- 预计集数/时长
- 核心看点
- 对标作品
- 受众群体
- 风格偏好

---

#### 🎯 节点2：题材定位
**负责 Agent**：策划师 Agent

**工作内容**：
- 精确的题材定位
- 核心卖点分析（3-5个）
- 商业价值评估
- 受众画像
- 市场竞争分析

**输出**：
- 题材定位报告
- 市场分析报告
- 商业价值评估

---

#### 🌍 节点3：世界观与人设
**负责 Agent**：世界观架构师 Agent + 人设师 Agent

**工作内容**：
- 构建完整世界观设定
- 创建核心人物档案

**输出**：
- 世界观设定：
  - 时间背景与历史
  - 社会结构
  - 文化体系
  - 地理环境
  - 特殊规则
- 人物档案：
  - 角色定位
  - 性格特征
  - 核心动机
  - 背景故事
  - 人物关系
  - 成长弧光

---

### 阶段二：大纲创作（节点4-5）

#### 📝 节点4：核心大纲
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 采用三幕式结构创作核心大纲
- 标注关键冲突、反转、伏笔
- 规划人物弧光节点

**输出**：
- 第一幕：铺垫、激励事件、情节点1
- 第二幕：中点、一无所有时刻、灵魂黑夜
- 第三幕：高潮、结局

---

#### ✅ 节点5：大纲校验
**负责 Agent**：剧本医生 Agent

**工作内容**：
- 人设一致性检查
- 剧情逻辑检查
- 格式规范检查

**输出**：
- 大纲审查报告
- 修改建议

---

### 阶段三：剧本创作（节点6-7）

#### 📚 节点6：分集/分场大纲
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 将核心大纲拆解为分集大纲
- 创建详细的分场大纲
- 规划伏笔设置

**输出**：
- 分集概述（每集核心事件）
- 分场大纲（每场场景的目标、冲突、节奏）
- 伏笔设置计划

---

#### ✍️ 节点7：剧本正文生成
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 按标准剧本格式生成正文
- 确保人物言行与人设一致
- 每场都有明确目标和冲突

**输出**：
- 标准格式剧本正文：
  - 场景标头（场号、时间、地点）
  - 场景描述
  - 人物对话
  - 动作描写
  - 情感提示

---

### 阶段四：终稿校验（节点8）

#### 🔍 节点8：终稿校验
**负责 Agent**：剧本医生 Agent

**工作内容**：
- 人设一致性检查
- 剧情逻辑检查
- 伏笔回收检查
- 格式标准化检查

**输出**：
- 终稿审查报告
- 质量评分
- 修改建议

---

## 🎯 完整流程图

```
需求解析 → 题材定位 → 世界观与人设 → 核心大纲 → 大纲校验
                                                         ↓
分集/分场大纲 → 剧本正文生成 → 终稿校验
```

## 📊 各 Agent 职责

| Agent | 负责节点 | 核心职责 |
|-------|----------|----------|
| 策划师 Agent | 1, 2 | 需求解析、题材定位、市场分析 |
| 世界观架构师 Agent | 3 | 世界观构建 |
| 人设师 Agent | 3 | 人物档案创建 |
| 主笔编剧 Agent | 4, 6, 7 | 大纲创作、剧本生成 |
| 剧本医生 Agent | 5, 8 | 质量审查、逻辑检查 |

## ✅ 8个节点说明

### 前期准备（3个节点）
- 节点1：需求解析 - 明确创作需求
- 节点2：题材定位 - 确定市场定位
- 节点3：世界观与人设 - 构建故事基础

### 大纲创作（2个节点）
- 节点4：核心大纲 - 创作故事框架
- 节点5：大纲校验 - 审查大纲质量

### 剧本创作（2个节点）
- 节点6：分集/分场大纲 - 详细拆解
- 节点7：剧本正文生成 - 生成剧本

### 终稿校验（1个节点）
- 节点8：终稿校验 - 全面质量检查

---

## 💡 创作流程特点

### 1. 分步确认
每个节点完成后都会暂停，等待用户确认后再进行下一步。

### 2. 质量把控
关键节点（5、8）有质量审查，确保创作质量。

### 3. 灵活调整
用户可以在任何节点提出修改意见，灵活调整创作方向。

---

## 🎬 总结

这个8节点的创作流程，确保了：
- ✅ 创意完整落地
- ✅ 结构清晰合理
- ✅ 人物立体饱满
- ✅ 质量全面把控

流程简洁高效，专注于剧本创作本身！
