# 剧本创作工作流 - 详细流程说明

## 🎬 完整创作流程（11节点版本）

### 阶段一：前期准备（节点1-3）

#### 📋 节点1：需求解析
**负责 Agent**：策划师 Agent

**工作内容**：
- 解析用户输入的创作需求
- 提取核心要素
- 初步确定创作方向

**输出**：
- 题材类型
- 预计集数/时长
- 核心看点
- 对标作品
- 受众群体
- 风格偏好

---

#### 🎯 节点2：题材定位
**负责 Agent**：策划师 Agent + 制片顾问 Agent

**工作内容**：
- 精确的题材定位
- 核心卖点分析（3-5个）
- 商业价值评估
- 受众画像
- 市场竞争分析
- 合规性初筛

**输出**：
- 题材定位报告
- 市场分析报告
- 初步合规建议

---

#### 🌍 节点3：世界观与人设
**负责 Agent**：世界观架构师 Agent + 人设师 Agent

**工作内容**：
- 构建完整世界观设定
- 创建核心人物档案

**输出**：
- 世界观设定：
  - 时间背景与历史
  - 社会结构
  - 文化体系
  - 地理环境
  - 特殊规则
- 人物档案：
  - 角色定位
  - 性格特征
  - 核心动机
  - 背景故事
  - 人物关系
  - 成长弧光

---

### 阶段二：大纲创作（节点4-5）

#### 📝 节点4：核心大纲
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 采用三幕式结构创作核心大纲
- 标注关键冲突、反转、伏笔
- 规划人物弧光节点

**输出**：
- 第一幕：铺垫、激励事件、情节点1
- 第二幕：中点、一无所有时刻、灵魂黑夜
- 第三幕：高潮、结局

---

#### ✅ 节点5：大纲校验
**负责 Agent**：剧本医生 Agent + 合规专员 Agent

**工作内容**：
- 人设一致性检查
- 剧情逻辑检查
- 合规性审查
- 格式规范检查

**输出**：
- 大纲审查报告
- 合规审查报告
- 修改建议

---

### 阶段三：剧本创作（节点6-7）

#### 📚 节点6：分集/分场大纲
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 将核心大纲拆解为分集大纲
- 创建详细的分场大纲
- 规划伏笔设置

**输出**：
- 分集概述（每集核心事件）
- 分场大纲（每场场景的目标、冲突、节奏）
- 伏笔设置计划

---

#### ✍️ 节点7：剧本正文生成
**负责 Agent**：主笔编剧 Agent

**工作内容**：
- 按标准剧本格式生成正文
- 确保人物言行与人设一致
- 每场都有明确目标和冲突

**输出**：
- 标准格式剧本正文：
  - 场景标头（场号、时间、地点）
  - 场景描述
  - 人物对话
  - 动作描写
  - 情感提示

---

### 阶段四：审查与建议（节点8-10）⭐ 重点

#### 🔍 节点8：终稿校验
**负责 Agent**：剧本医生 Agent

**工作内容**：
- 人设一致性检查
- 剧情逻辑检查
- 伏笔回收检查
- 格式标准化检查

**输出**：
- 终稿审查报告
- 质量评分
- 修改建议

---

#### ✅ 节点9：合规审查 ⭐ 过审步骤
**负责 Agent**：合规专员 Agent

**工作内容**：
- 内容合规性检查
- 政策法规符合性审查
- 敏感内容识别
- 风险评估

**审查内容**：
1. **政治敏感度**
   - 是否涉及敏感政治议题
   - 是否符合当前政策导向

2. **价值观导向**
   - 是否传递正面价值观
   - 是否符合主流审美

3. **内容尺度**
   - 暴力内容检查
   - 色情内容检查
   - 恐怖内容检查
   - 粗俗语言检查

4. **社会影响**
   - 是否引发负面舆情
   - 是否有不良引导

5. **版权问题**
   - 是否涉及侵权
   - 是否需要授权

6. **其他合规**
   - 广播电视播出标准
   - 网络视听节目审核标准
   - 涉案剧管理规定

**输出**：
- 合规审查报告
- 风险等级（低/中/高）
- 敏感内容清单
- 修改建议
- **通过/不通过**结论

**审查标准**：
- ✅ 通过：无敏感问题，可直接送审
- ⚠️ 有条件通过：有轻微问题，修改后可送审
- ❌ 不通过：有严重问题，需要大幅修改

---

#### 🎬 节点10：制片建议 ⭐ 拍摄建议
**负责 Agent**：制片顾问 Agent

**工作内容**：
- 制作可行性评估
- 成本预算估算
- 拍摄周期规划
- 技术要求分析
- 市场推广建议

**建议内容**：

**1. 制作可行性**
- 拍摄难度评估
- 特效需求分析
- 场景需求评估
- 道具服装需求
- 选角建议

**2. 成本预算**
- 制作成本估算
- 演员成本
- 后期制作成本
- 宣发成本
- 总预算建议

**3. 拍摄周期**
- 前期筹备周期
- 拍摄周期
- 后期制作周期
- 总周期规划

**4. 技术要求**
- 摄影器材要求
- 灯光设备要求
- 后期技术要求
- 特效制作要求

**5. 市场推广**
- 目标受众分析
- 宣发策略建议
- 发行渠道建议
- 收益预期

**6. 风险提示**
- 制作风险
- 市场风险
- 合规风险
- 其他风险

**输出**：
- 制片建议报告
- 可行性评估（可行/有风险/不可行）
- 预算方案
- 周期规划
- 市场推广方案
- 风险提示

---

### 阶段五：修改迭代（节点11）

#### 🔄 节点11：修改迭代
**负责 Agent**：主笔编剧 Agent + 剧本医生 Agent

**工作内容**：
- 根据合规审查意见修改
- 根据制片建议调整
- 保持整体连贯性

**输出**：
- 修改后的剧本
- 修改记录
- 版本号更新

---

## 🎯 完整流程图

```
需求解析 → 题材定位 → 世界观与人设 → 核心大纲 → 大纲校验
                                                         ↓
分集/分场大纲 → 剧本正文生成 → 终稿校验 → 合规审查 ⭐ → 制片建议 ⭐
                                                         ↓
                                                   修改迭代
```

## ✅ 关键节点说明

### ⭐ 节点9：合规审查（过审步骤）

**重要性**：★★★★★

**为什么重要**：
1. 政策法规要求
2. 避免审查不通过
3. 减少修改成本
4. 降低播出风险

**审查流程**：
1. 全面扫描剧本内容
2. 对照审查标准
3. 识别敏感问题
4. 评估风险等级
5. 给出修改建议
6. 得出审查结论

### ⭐ 节点10：制片建议（拍摄建议）

**重要性**：★★★★☆

**为什么重要**：
1. 评估制作可行性
2. 控制成本预算
3. 规划拍摄周期
4. 提高制作效率
5. 增强市场竞争力

**建议流程**：
1. 分析剧本需求
2. 评估制作难度
3. 估算成本预算
4. 规划拍摄周期
5. 分析技术要求
6. 制定推广策略

---

## 💡 使用建议

### 1. 合规审查前置
- 在大纲阶段就进行合规初筛
- 减少后期修改成本
- 避免反复调整

### 2. 制片建议参考
- 根据制片建议调整剧本
- 降低制作难度
- 控制成本预算

### 3. 迭代优化
- 根据审查和建议反复修改
- 确保质量达标
- 提高通过率
